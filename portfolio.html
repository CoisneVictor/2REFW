Portfolio

Mon portfolio répertorie tous les projets de développement que j'ai pu réaliser jusqu'à présent. Vous y trouverez des sites web, des applications, des jeux vidéos et des jeux de plateau. Chaque projet devait être réalisé dans un temps imparti qui varier entre deux et trois semaines. (Excepté City Idle qui est un projet de fin d'année et Flappy Bird like qui est un projet personnel).

Supemon

Supemon est un Pokemon like ou j'ai dû développer un système de combat pokémon et de gestion de ressources dans le cadre de l'apprentissage du langage C à SUPINFO.
Affichage : CLI (Command Line Interface)
Langage : C

Jeu du cavalier

Le jeu du cavalier est un jeu où les joueurs se déplacent comme le cavalier des échecs en laissant une croix sur leur passage. Lorsque le nombre de croix choisit est aligné, le joueur remporte la partie.
Affichage : Tkinter
Langage : Python

Flappy Bird like

Le joueur incarne un oiseau devant passer entre les tuyaux. Le jeu se déroule à l’infini jusqu’à que le joueur heurte un tuyau.
Affichage : Unity Game Tab
Langage : C#

City Idle

Projet en cours de développement. City Idle est un jeu de gestion où le joueur gère une population cherchant à évoluer au travers des âges.
Affichage : Unity Game Tab
Langage : C#

Site de streaming

Ce projet était centré sur l’utilisation de JavaScript. J’ai utilisé l’API IMDB afin de créer un catalogue pour un site de streaming avec une page descriptive qui s’adapte au film choisit.
Affichage : Web
Langage : HTML, CSS et JavaScript
Démonstration : Site de streaming

Location d'immobilier

Ce projet a pour objectif de faire de la réservation d’immobilier. J’ai crée plusieurs classes et je les ai fait interagir entre-elles pour le réaliser.
Affichage : CLI
Langage : TypeScript

Gestion de base de donnée

Ici j’ai simulé la gestion d’une base de données pour une école. J’ai créé des requêtes afin de remplir les tables avec les informations de la base de données.

Langage : SQL

Infrastructure réseau

Lors de ce projet, j’ai simulé l’expansion d’une entreprise vers l’étranger et la création de l’administration informatique de l’entreprise.

Gestion de magasin

Ce projet a pour but de créer un système de gestion de magasins et d’utilisateurs. Les utilisateurs ont la possibilité de modifier le stock d’un magasin et les administrateurs peuvent créer des magasins et gérer les utilisateurs.

Affichage : Java et SQL
